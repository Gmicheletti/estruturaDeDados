Lista de exercícios 03 – Estruturas de Dados
OK 1. Implemente a classe ListaEncadeada para armazenar valores inteiros referências
para ligar os elementos da lista.
OK 2. Crie um método para criar uma lista vazia.
OK 3. Crie um método responsável por inserir um elemento no início da lista.
OK 4. Crie um método responsável por inserir um elemento no fim da lista.
OK 5. Crie um método responsável por inserir um elemento em uma posição específica
da lista.
OK 6. Crie um método responsável por remover um elemento no início da lista.
OK 7. Crie um método responsável por remover um elemento no fim da lista.
OK 8. Crie um método responsável por remover um elemento em uma posição
específica da lista.
OK 9. Crie um método responsável por remover um elemento específico da lista.
OK 10. Crie um método para exibir o conteúdo de uma lista.
OK 11. Crie um método para pesquisar por um elemento específico em uma lista e
informar, caso o elemento exista, a posição na qual ele está armazenado.
OK 12. Crie um método que retorne o número de elementos existentes na lista.
