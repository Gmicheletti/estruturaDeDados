Lista de exercícios 02 – Estruturas de Dados
1. OK Implemente a classe ListaEncadeada utilizando um arranjo de inteiros para
armazenar os elementos da lista.
2. OK Crie um método para criar uma lista vazia.
3. OK Crie um método responsável por inserir um elemento no início da lista.
4. OK Crie um método responsável por inserir um elemento no fim da lista.
5. OK Crie um método responsável por inserir um elemento em uma posição específica
da lista.
6. OK Crie um método responsável por remover um elemento no início da lista.
7. OK Crie um método responsável por remover um elemento no fim da lista.
8. OK Crie um método responsável por remover um elemento em uma posição
específica da lista.
9. OK Crie um método responsável por remover um elemento específico da lista.
10. OK Crie um método para exibir o conteúdo de uma lista.
11. OK Crie um método para pesquisar por um elemento específico em uma lista e
informar, caso o elemento exista, a posição na qual ele está armazenado.
12. OK Crie um método que retorne o número de elementos existentes na lista.
